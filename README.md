[![CI](https://github.com/egormikhajlov/mock-data-generator/actions/workflows/ci.yml/badge.svg?branch=main)](https://github.com/egormikhajlov/mock-data-generator/actions/workflows/ci.yml)

**Генератор случайных персональных данных** — лабораторная работа №3  
Автор: **Егор Михайлов**  
Дата: ноябрь 2025

## Что делает программа

Приложение генерирует случайные данные о вымышленных людях и сохраняет их в формате **JSON**:

- Имя и фамилия (на русском языке)  
- Возраст (от 18 до 77 лет)  
- Email, телефон и адрес

Вы можете указать:
- Сколько записей создать (по умолчанию — 10)  
- Имя выходного файла (по умолчанию — `output.json`)

Результат сохраняется в файл на вашем компьютере.

Технологии: **Java 17 + Gradle + Jackson + JUnit 5**

## Полная инструкция по запуску (для тех, кто делает это впервые)

Для начала необходимо открыть командную строку. Для этого нажмите комбинацию клавиш:

- **WIN + R**

И в открывшемся окне напишите:

- **cmd**

Нажмите Enter.

# 1. Создаём папку и клонируем проект

В открывшемся терминале выполните по очереди:

- `mkdir mock-data-project`  
- `cd mock-data-project`  
- `git clone https://github.com/egormikhajlov/mock-data-generator.git`  
- `cd mock-data-generator`

# 2. Собираем проект

Выполните следующую команду:

- `gradlew build`

Дождитесь сообщения:  
`BUILD SUCCESSFUL`

# 3. Запускаем приложение

Теперь запустите программу:

- `gradlew run`

После запуска вы увидите в командной строке:

```
Generating 10 random records...
Data successfully written to: output.json
```

# 4. Где найти результат?

Файл с данными создаётся в папке проекта.  
Откройте **Проводник** и перейдите по пути: Ваш\путь\mock-data-generator\app

Кликните по файлу дважды — он откроется в браузере или Блокноте.
